Poly-filters

Useful filters for Polymer expressions

Filters:

  • Uppercase
  • Lowercase
  • Trim
  • Round
  • Reverse
  • Length
  • Parsing link
  • Date
  • Array filter

Uppercase

{{ value | uppercase }}

Lowercase

{{ value | lowercase }}

Trim

{{ value | trim }}

Round

{{ value | round }}

Reverse

{{ value | reverse }}

Length

{{ value | length }}

Parse link

{{ value | link('pathname') }}

Date

{{ value | date('yy-MM-dd HH:mm t') }}

Array filter

{{ [
    {
        name: 'Addy Osmani'
    },
    {
        name: 'Paul Irish'
    },
    {
        name: 'Ryan Dahl'
    },
    {
        name: 'Sara Chipps'
    }
] | arrayFilter('name', value) }}
